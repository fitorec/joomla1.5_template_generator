Generador de plantillas para joomla1.5
==========================================================================================

Caracteristicas principales
-------------------------------
 - No requiere instalación.
 - Multi-idiomas
   - Ingles
   - Español
 - Funciona sobre
   - Linux
   - Mac
   - *nix
   - No Windows(lo siento)

Como utilizar?
-----------------------------
>**Para correr hacer click sobre el archivo `template_generator`**
